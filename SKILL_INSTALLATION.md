## 技能功能

### 设计思维
- **目的**: 界面解决的问题，谁使用它
- **调性**: 选择极端风格（极简、极繁、复古未来、有机自然、奢华精致等）
- **约束**: 技术要求（框架、性能、可访问性）
- **差异化**: 什么让它难忘？什么是用户会记住的那一点？

### 前端美学指南

#### 排版
- 选择美丽、独特、有趣的字体
- 避免通用字体（Arial、Inter）
- 选择能提升前端美学的独特选择
- 将独特的展示字体与精致的正文字体配对

#### 颜色与主题
- 坚持一致的审美
- 使用 CSS 变量保持一致性
- 主色调配锐利强调色胜过胆小的均匀调色板

#### 动效
- 使用动画实现效果和微交互
- HTML 优先使用纯 CSS 解决方案
- 聚焦高影响力时刻：一个精心编排的页面加载

#### 空间构图
- 意想不到的布局
- 不对称
- 重叠
- 对角线流动
- 打破网格的元素

#### 背景与视觉细节
- 创造氛围和深度
- 渐变网格、噪点纹理、几何图案
- 分层透明度、戏剧性阴影、装饰边框

### 禁止使用的通用"AI 平庸"美学
- ❌ 过度使用的字体家族（Inter、Roboto、Arial、系统字体）
- ❌ 陈词滥调的配色方案（特别是白色背景上的紫色渐变）
- ❌ 可预测的布局和组件模式
- ❌ 缺乏情境特定特征的千篇一律设计
